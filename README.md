# FYP2526-ZhengDong-Mapping-research-at-XJTLU

This project is an interactive dashboard for faculty academic analysis.

## Main functions
- Data scraping and cleaning
- Topic analysis
- Collaboration network analysis
- Search and visualization dashboard

## Main files
- `app.py`: application entry
- `pages/`: dashboard pages
- `utils/`: analysis and helper functions
- `layouts/`: shared layout components
- `assets/`: CSS and JS files

## Note
Large data files are not included in this repository.
Scraped datasets are shared separately.
